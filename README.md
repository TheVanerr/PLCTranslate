# Excel Görüntüleyici (Basit)

Bu küçük proje, bir Excel dosyası yüklemenizi ve sayfa üzerinde görüntülemenizi sağlar. HTML/CSS/JS ile yazıldı ve SheetJS (`xlsx`) CDN ile çalışır — sunucuya ihtiyaç yoktur.

Dosyalar

- `index.html` — Ana sayfa
- `style.css` — Basit stiller
- `app.js` — Excel okuma ve görüntüleme kodu
- `netlify.toml` — Netlify için basit publish ayarı

Kullanım (yerel):

1. Klasördeki dosyaları açın (ör. çift tıklayarak `index.html`).
2. Bir `.xlsx` veya `.xls` dosyasını seçin veya sürükleyin.

Deploy (Netlify):

1. Bu klasörü bir git deposuna koyun (isteğe bağlı).
2. Netlify hesabı oluşturun/oturum açın.
3. "New site from Git" ile depoyu bağlayın veya "Deploy > Drag and drop" ile klasörü sürükleyin.
